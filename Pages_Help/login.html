2. Login


Sebelum memulai aplikasi, user diharuskan untuk login terlebih dahulu untuk menjaga keotentikan user, halaman login akan tampil seperti gambar berikut:





masukkan username dan password anda untuk login ke website Adaro Land Management System


Tahapan login yang harus dilakukan pengguna adalah sebagai berikut:

1.        Masukkan username yang telah didaftarkan, seperti gambar dibawah ini.

2.        Masukkan password, seperti gambar dibawah ini.

3.        Setelah mengisi kolom email dan password, maka klik buttonsign in, apabila username dan password sesuai maka user akan masuk kedalam halaman utama aplikasi sedangkan jika user mengisi email dan password  yang tidak sesuai maka user harus mengulangi proses login.
