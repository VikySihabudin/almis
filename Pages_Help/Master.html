3.11.1 Master Identitas


Keterangan halaman halaman Master Identitas:

  1. User  melakukan pencarian data ListMaster Identitas berdasarkan pada list drop down nama perusahaan.


  • Pilih salah satu dari list dropdown nama perusahaan.
  • Maka akan ditampilkan listMaster Identitas berdasarkan perusahaan yang dipilih


2. User melakukan pencarian data ListMaster Identitas berdasarkan periode pada list drop down.


    • Pilih salah satu dari list dropdown periode.
    • Maka akan ditampilkan listMaster Identitas berdasarkan periode yang dipilih


3. User  melakukan pencarian data ListMaster Identitas berdasarkan Date Awal dan Date Akhir, dengan  mengisi kedua kolom kemudian menekan button Search. Jika salah satu kolom date tidak diisi maka akan muncul warning. dan data tidak akan ditampilkan.


  1. Add New, untuk menampilkan halaman tambah data Master Identitas.


Langkah-langkah Add New Data Master Identitas:

  • Pilih nama perusahaan
  • Masukan nomor identitas dan pilih tanda identitas.
  • Masukkan Nama, Jenis kelamin, Tanggal Lahir, Pekerjaan, Nomor Induk Pegawai, Alamat dan Nomor Handphone.
  • Isi Data Pernikahan, Anak dan Orang Tua [Pilihan].
  • Kemudian masukan file foto, bisa dengan menekan button Ambil Foto atau pilih file dari komputer dengan menekan button Select File.
  • Setelah mengisi semua maka tekan button Save, maka data akan tersimpan dan akan ada pesan “Data Berhasil Di Simpan”.
  • Tekan Button Cancel untuk membatalkan proses simpan dan untuk menutup halaman Add New Master Identitas.


  1. List Data, menampilkan data yang tersimpan pada database.
  2. Pencarian berdasarkan isi kolom, user memasukkan data pada kolom yang dimaksud, apabila data yang dicari ada maka data akan tampil pada list.
  3. Aksi
  1. View, menampilkandetail data Master Identitasyang di pilih.

  • Tekan button Cancel untuk menutup halaman View Master Identitas.
  1. Edit, menampilkan halaman untuk merubah data Master Identitasyang dipilih sebelumnya.

  • Setelah user mengganti data yang perlu di edit maka klik button save untuk menyimpan data.
  • Pilih cancel untuk membatalkan proses edit dan menutup halaman edit Master Identitas.
  1. Delete
    • Tekan button delete, kemudian akan muncul konfirmasi hapus.
    • Tekan OK untuk menghapus data atau TekanCancel untuk membatalkan proses hapus
