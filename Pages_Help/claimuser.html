3.6.1 Claim User

Untuk menuju ke halaman Claim User, tahapan yang dilakukan adalah sebagai berikut:


  • Klik menu Land Litigation > Claim User


  • Akan tampil halaman Claim User seperti gambar berikut



Keterangan Halaman Claim User:

  1. User  melakukan pencarian data ListClaim User berdasarkan pada list drop down nama perusahaan.


  • Pilih salah satu dari list dropdown nama perusahaan.
  • maka akan ditampilkan listClaim User berdasarkan perusahaan yang dipilih


2. User melakukan pencarian data ListClaim User  berdasarkan periode pada list drop down.


  • Pilih salah satu dari list dropdown periode.
  • maka akan ditampilkan list Claim Userberdasarkan periode yang dipilih


3. User  melakukan pencarian data List Claim User berdasarkan Date Awal dan Date Akhir, dengan  mengisi kedua kolom kemudian menekan button Search. Jika salah satu kolom date tidak diisi maka akan muncul warning. dan data tidak akan ditampilkan.

4. Button Add New, digunakan untuk menambahkan data Claim User.


Tekan Button Add New maka akan tampil halaman tambah data Claim User seperti dibawah ini.


5. Pencarian berdasarkan isi kolom, user memasukkan data pada kolom yang dimaksud, apabila data yang dicari ada maka data akan tampil pada list.

      6. List Data menampilkan data yang terdapat pada database.
