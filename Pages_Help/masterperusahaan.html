3.11.6 Master Perusahaan


Untuk menuju ke halaman Master Perusahaan, tahapan yang dilakukan adalah sebagai berikut:


  • Klik menu Admin Tools > Master Perusahaan


  • Akan tampil halaman Master Perusahaan seperti gambar berikut



Keterangan halaman halaman Master Perusahaan:

  1. User  melakukan pencarian data List Master Perusahaan berdasarkan pada list drop down nama perusahaan.
  • Pilih salah satu dari list dropdown nama perusahaan.
  • Maka akan ditampilkan list Master Perusahaan berdasarkan perusahaan yang dipilih


2. User melakukan pencarian data List Master Perusahaan berdasarkan periode pada list drop down.


    • Pilih salah satu dari list dropdown periode.
    • Maka akan ditampilkan list Master Perusahaanberdasarkan periode yang dipilih


3. User  melakukan pencarian data List Master Perusahaan berdasarkan Date Awal dan Date Akhir, dengan  mengisi kedua kolom kemudian menekan button Search.

Jika salah satu kolom date tidak diisi maka akan muncul warning. dan data tidak akan ditampilkan.


  1. Button Add New , untuk menampilkan halaman tambah data Master Perusahaan.



Langkah-langkah tambah data Master Perusahaan:


  • Masukan data pada kolom ID Perusahaan. ID tidak boleh sama dengan yang sudah ada sebelumnya. Jika sama maka sistem akan memberikan pesan peringatan.
  • Masukan nama pada nama perusahaan.
  • Masuka keterangan tentang perusahaan pada kolom keterangan.
  • Tekan button Save, jika data yang dimasukkan sudah benar maka sistem akan menyimpan data dan akan memberi pesan bawa data telah tersimpan.
  • Tekan button Cancel untuk membatalkan proses dan menutup halaman Tambah Data Master Perusahaan.


  1. List Data, menampilkan data yang tersimpan pada database.
  2. Pencarian berdasarkan isi kolom, user memasukkan data pada kolom yang dimaksud, apabila data yang dicari ada maka data akan tampil pada list.
  3. Aksi
  1. View, menampilkandetail data Master Perusahaan yang di pilih.



  • Tekan button Cancel untuk menutup halaman View Master Perusahaan.


  1. Edit, menampilkan halaman untuk merubah data Master Perusahaan yang dipilih sebelumnya.



  • Setelah user mengganti data yang perlu di edit maka klik button save untuk menyimpan data.
  • Pilih cancel untuk membatalkan proses edit dan menutup halaman edit User Management.


  1. Delete, untuk menampilkan halaman delete data Master Perusahaan.



    • Tekan button delete, kemudian akan muncul konfirmasi hapus.
    • Tekan button OK pada konfirmasi untuk menghapus data atau Tekanbutton Cancel untuk membatalkan proses hapus.
    • Tekan button Cancel untuk menutup halaman Delete Master Perusahaan.
