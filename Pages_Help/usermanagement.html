3.11.  3.11.2 User Management


Untuk menuju ke halaman User Management, tahapan yang dilakukan adalah sebagai berikut:


  • Klik menu Admin Tools  > User Management


  • Akan tampil halaman User Management seperti gambar berikut



Keterangan halaman halaman User Management:

  1. User  melakukan pencarian data ListUser Management berdasarkan pada list drop down nama perusahaan.


  • Pilih salah satu dari list dropdown nama perusahaan.
  • Maka akan ditampilkan listUser Management berdasarkan perusahaan yang dipilih


2. User melakukan pencarian data ListUser Managementberdasarkan periode pada list drop down.


    • Pilih salah satu dari list dropdown periode.
    • Maka akan ditampilkan listUser Management berdasarkan periode yang dipilih


3. User  melakukan pencarian data ListUser Management berdasarkan Date Awal dan Date Akhir, dengan  mengisi kedua kolom kemudian menekan button Search. Jika salah satu kolom date tidak diisi maka akan muncul warning. dan data tidak akan ditampilkan.


  1. Button Tambah , untuk menampilkan halaman tambah data User Management.

Langkah-langkah tambah data User Management:

  • Masukan data User Id. User Id tidak boleh sama dengan yang sudah ada sebelumnya. Sistem akan memberi pesan jika User ID sudah digunakan.
  • Masukan Nama identitas.
  • Masukan alamat email.
  • Pilih Group pada dropdown Group.
  • Pilih status active pada dropdown status.
  • Tekan button Save, jika data yang dimasukkan sudah benar maka sistem akan menyimpan data dan akan memberi pesan bawa data telah tersimpan.
  • Tekan button Cancel untuk membatalkan proses dan menutup halaman Tambah Data User Management.



  1. List Data, menampilkan data yang tersimpan pada database.
  2. Pencarian berdasarkan isi kolom, user memasukkan data pada kolom yang dimaksud, apabila data yang dicari ada maka data akan tampil pada list.
  3. Aksi
  1. View, menampilkandetail data User Management yang di pilih.

  • Tekan button Cancel untuk menutup halaman ViewUser Management.
  1. Edit, menampilkan halaman untuk merubah data User Management yang dipilih sebelumnya.

  • Setelah user mengganti data yang perlu di edit maka klik button save untuk menyimpan data.
  • Pilih cancel untuk membatalkan proses edit dan menutup halaman editUser Management.


  1. Delete
    • Tekan button delete, kemudian akan muncul konfirmasi hapus.
    • Tekan button OK pada konfirmasi untuk menghapus data atau Tekanbutton Cancel untuk membatalkan proses hapus.
    • Tekan button Cancel untuk menutup halaman Delete User Management.
