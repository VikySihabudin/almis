3.11.3 Group Management

Keterangan halaman halaman Group Management:

  1. User  melakukan pencarian data ListGroup Management berdasarkan pada list drop down nama perusahaan.
  • Pilih salah satu dari list dropdown nama perusahaan.
  • Maka akan ditampilkan listGroup Management berdasarkan perusahaan yang dipilih


2. User melakukan pencarian data ListGroup Management berdasarkan periode pada list drop down.


    • Pilih salah satu dari list dropdown periode.
    • Maka akan ditampilkan listGroup Management berdasarkan periode yang dipilih


3. User  melakukan pencarian data ListGroup Management berdasarkan Date Awal dan Date Akhir, dengan  mengisi kedua kolom kemudian menekan button Search. Jika salah satu kolom date tidak diisi maka akan muncul warning. dan data tidak akan ditampilkan.


  1. ButtonAdd New , untuk menampilkan halaman tambah data Group Management.

Langkah-langkah tambah data GroupManagement:

  • Masukan Nama Group
  • Pilih Group Lanjut pada dropdown.
  • Pilih Group Except pada dropdown.
  • Tekan button Save, jika data yang dimasukkan sudah benar maka sistem akan menyimpan data dan akan memberi pesan bawa data telah tersimpan.
  • Tekan button Cancel untuk membatalkan proses dan menutup halaman Tambah Data GroupManagement.


  1. List Data, menampilkan data yang tersimpan pada database.
  2. Pencarian berdasarkan isi kolom, user memasukkan data pada kolom yang dimaksud, apabila data yang dicari ada maka data akan tampil pada list.
  3. Aksi
  1. View, menampilkandetail data Group Management yang di pilih.

  • Tekan button Cancel untuk menutup halaman ViewGroup Management.


  1. Edit, menampilkan halaman untuk merubah data Group Management yang dipilih sebelumnya.

  • Setelah user mengganti data yang perlu di edit maka klik button save untuk menyimpan data.
  • Pilih cancel untuk membatalkan proses edit dan menutup halaman editUser Management.


  1. Delete, untuk menampilkan halaman delete data Group Management.

    • Tekan button delete, kemudian akan muncul konfirmasi hapus.
    • Tekan button OK pada konfirmasi untuk menghapus data atau Tekanbutton Cancel untuk membatalkan proses hapus.
    • Tekan button Cancel untuk menutup halaman Delete Group Management.
